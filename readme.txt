Automatically complete fields for files attached to the posts.

== Description ==
To automatically complete fields for files attached to the posts. The image appears description, caption and an alt attribute. As well as updated title (file name, and the name of the library and multimedia). All fields will have the same value as the name of the parent post.
<br><br>
Автоматически заполняет поля для изображений, прикреплённых к записям. У изображения появится описание, заголовок и атрибут alt. А так же название (не имя файла, а название элемента библиотеки мультимедиа). Во все поля будет скопировано название записи, к которой прикреплено изображение.

== Installation ==
1) Upload files to the "/wp-content/plugins/" directory.

2) Activate the plugin through the "Plugins" menu in WordPress.


1) Загрузите файлы в папку "/wp-content/plugins/".

2) Активируйте плагин в меню "Плагины".

== Frequently Asked Questions ==
= Why is it necessary? =
To display beautiful names in the slide show and similar plugins.

= Зачем это нужно? =
Чтобы отображались красивые названия в слайд-шоу и аналогичных плагинах.

= How to manually fill in the fields for the file, and disable changing them? =
In the administrative part of the site, select "Media". In the right column ("Autorename") for the desired file, click "Turn off".
If this column there, in the top right corner of the screen and click "display Settings" and enable the column "Autorename".

= Как вручную заполнить поля для файла и запретить менять их? =
В административной части сайта выберите пункт "Медиафайлы". В правой колонке ("Автозаполнение") для нужного файла перейдите по ссылке "Исключить".
Если такой колонки нет, в правом верхнем углу экрана нажмите на кнопку "Настройки экрана" и включите отображение колонки "Автозаполнение".
